BluBop: Earthquake Detection Robot

  • Overview: BluBop, programmed in C, is a joystick-controlled robot that stops for vibrations, ensuring safety during earthquakes, and performs a "dance" after detecting repeated vibrations.
  • Implementation: Uses ADC and PWM for joystick-driven motor control and vibration detection, with custom libraries and logic for precise signal handling.
  • Hardware Integration: Features a joystick for directional input, a vibration sensor for safety triggers, and servo motors controlled via PWM, all powered through a single microcontroller board.
  • Testing: Verified joystick, motor, and vibration functionality through physical trials and debugging of ADC/PWM configurations.

Sketchify: ML Coloring Book

  • Overview: Sketchify is an ML-powered system that converts images into coloring book pages using segmentation, edge detection, smoothing, and SVG conversion for infinite resolution.
  • Advanced Segmentation: Uses Meta’s Segment Anything Model to isolate subjects from images without additional training.
  • Edge Detection and Smoothing: Combines Canny and TEED detection with OpenCV techniques to refine edges by denoising and connecting contours.
  • SVG Conversion: Employs VTracer to transform refined edges into smooth, scalable vector paths with infinite resolution.

MathSearch

  • Overview: MathSearch is a scalable search engine that finds and matches mathematical equations in academic papers using advanced machine learning techniques.
  • Core Features: Utilizes YOLOv5 for equation detection, VGG-16 for similarity scoring, and a user-friendly interface supporting live LaTeX queries and multi-PDF uploads.
  • Implementation: Leverages an S3-based queueing system, intermediaries for seamless integration, and bounding box visualizations to display detected equations on PDFs.
  • User Interaction: Provides an intuitive UI for uploading documents, rendering search results with bounding boxes, and navigating matched equations effortlessly.
